Transformer

Javascript plug that enables an element to be dragged, scaled and rotated.

Download .zip Download .tar.gz View on GitHub

Rotate, Scale and Translate

Using the transformer plugin enables you to translate, rotate and scale an HTML element. Try it out below!

Disable Features

Specific features of the plugin can be disabled at initialization or at runtime.

Listen to Transform Events

A register event listeners for the following events on a Transformable object: beforerotate, rotate, beforescale, scale, beforetranslate and translate.

The following example changes the color the box if it is rotated past 180 degrees.

Customize Interaction

Limit user interaction by canceling the before events. Example interaction would be to limit the size of a selection box or prevent it from moving past a bounding box.

In this example the box is limited to rotations between -45 and 45 degrees

Hide and Show the Selection Elements

Hide and show the selection elements without hiding the content it wraps. Double click on the box below toggle the tranformation elements.

Change Style of Selection Element

CSS file is compiled in SASS and has a theme sheet that can be adjusted easily to create a whole different feel.
